## Captured X Signal

Just out of curiosity, I let Claude attempt to port Loupe to Android using @skiptools. If you haven’t heard of Skip, it’s a tool that lets you produce native Android apps from a SwiftUI codebase. Loupe is written entirely in Swift and SwiftUI, so it should be a good match

This is the result after just one prompt (which was more or less like: port this app to Android using skip tools make no mistakes). The good news is that it actually builds and runs on device, the bad news is that pretty much nothing is functional yet. Claude recognised that it still needs to replace all the various native iOS frameworks for things like photos, location, and device motion with their Android equivalents, but for now it decided to defer all that work and just focus on getting the code to compile for Android

It’s still so surprising to me that this works at all. That said, I don’t think Loupe for Android is coming anytime soon since we have other stuff going on right now
