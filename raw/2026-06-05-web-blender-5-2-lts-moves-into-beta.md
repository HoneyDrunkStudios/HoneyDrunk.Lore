# Blender 5.2 LTS Is Now In Beta

Check out experimental Geometry Nodes-based physics for hair and cloth, a new Thin Wall mode for rendering translucent surfaces, Cycles texture cache that reduces memory usage, and much more.

We've been getting glimpses of what's coming in the next major release of Blender for the past few months, and now the time has come to download it and help make it stable for everyone. The new splash screen for Blender 5.2 LTS features a cave lion (Panthera spelaea) by [Joanna Kobierska](https://www.artstation.com/joanna_kobierska).

Before you download it, here are a few features you'll want to check out first:

[Cycles texture cache](https://80.lv/articles/blender-5-2-lts-introduces-new-cycles-texture-cache-system), which cuts memory use and startup time on texture-heavy scenes;- New Hair/Cloth dynamics, an exciting addition which we've covered on our site a
[few](https://80.lv/articles/blender-5-2-alpha-s-cloth-tearing-simulation-test)[times](https://80.lv/articles/iconic-ghost-in-the-shell-effect-recreated-with-blender-5-2-s-new-cloth-solver)already; - New Thin Wall mode in Principled BSDF to render leaves, paper, and other meshes with minimal thicknesses in a more photoreal way;

- Grease Pencil with a brand new Fill tool and updated brush library;
- LoopTools add-on features such as Circle, Space, and Flatten are now built into Blender;
- New Mesh Bevel node and Sample Sound node for extracting frequency data from audio, Geometry Bundles, Lists, support for Empty objects, and more in Geometry Nodes.

Also, support for remotely hosted asset libraries has been added, so you can browse and download assets without leaving Blender. Enable Allow Internet Access in the Preferences to access this functionality.

As always, users are encouraged to test the beta release and report any issues they encounter, helping the development team deliver a stable release on July 8th.

See the full list of new features in [Blender 5.2 LTS](https://developer.blender.org/docs/release_notes/5.2/) and download it [here](https://builder.blender.org/download/daily/).
