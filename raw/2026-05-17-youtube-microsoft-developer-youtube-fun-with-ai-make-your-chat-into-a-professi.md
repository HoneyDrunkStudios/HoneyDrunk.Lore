You are a professional makeup artist and counselor at a beauty salon. Based on the following information, create a "Beauty Salon Counseling Sheet Style" makeup diagnosis.
# Purpose
Please create a structure for an infographic that is easy to organize visually, especially for social media sharing.
# Output Format
- Structured and organized by headings.
- Detail level suitable for direct printing.
- Professional yet easy-to-understand English.
# Required Items
1. Face Type Analysis (Shape, features, impression)
2. Personal Color Tendency (Warm/Cool + Tone)
3. Suitable Makeup Direction (Using 3 keywords)
4. Makeup Proposal by Feature
- Foundation
- Eye Makeup
- Blush
- Lips
5. Makeup Don'ts (Common mistakes to avoid)
6. Glow-up Points (The 3 most important points)
7. Suggested Hair colors and styles
8. A short summary (A salon-style professional comment in handwriting)
# Layout Aesthetics
- Elegant and sophisticated layout, like beauty salon counseling materials.
- Minimalist design with effective use of white space.
- Sections divided into short segments (No long paragraphs).
- Light colors and high-end design. No dark mode.
# Input Information // Edit this section as you wish!
- Face Type (e.g., Round, Long, etc.)
- Face Features (e.g., Large eyes, low nose bridge, etc.)
- Skin Tone (e.g., Fair, yellowish, etc.)
- Desired Look (e.g., Sophisticated, K-pop style, etc.)
# Supplementary Notes
- Maintain the original facial features (No heavy editing/filtering), but remove background and replace with a salon-looking background
- The diagnosis result image should be created based on the face in the original image.
- Output as an image.
