# View Agentic Workflow configs in the Actions run summary

Back to changelog 
Improvement 
March 26, 2026 •

View Agentic Workflow configs in the Actions run summary 
You can now view your Agentic Workflow markdown configs directly in the GitHub Actions run summary for any agentic workflow. This update makes it easier to review runs by:
Reducing the need to switch between pages. 
Showing the exact configuration used when the workflow ran. 
For more information about GitHub Agentic Workflows, see the documentation . To get started, see the quickstart guide .
