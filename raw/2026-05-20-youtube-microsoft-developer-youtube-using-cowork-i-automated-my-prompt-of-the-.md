## Description

Every week I run a Prompt of the Week initiative with shoutouts to community contributors from our open source repo. I was doing the slide manually every single time. Not anymore.
I built a skill in Cowork that takes one GitHub link, fetches the metadata straight from the contribution, and generates a fully formatted PowerPoint slide. Done.
This skill is coming to the prompt library soon — stay tuned.
