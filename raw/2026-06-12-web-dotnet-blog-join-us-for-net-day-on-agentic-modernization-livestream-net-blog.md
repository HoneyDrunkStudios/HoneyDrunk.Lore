.NET Modernization

Join us June 16th for .NET Day on Agentic Modernization — explore agentic modernization with demos and AI strategies for .NET.

[RSVP now](https://aka.ms/dotnet/agentic-mod/rsvp)

[Jeffrey Fritz](https://devblogs.microsoft.com/dotnet/author/jeffreytfritzyahoo-com)

Principal Program Manager

If you’re modernizing .NET apps and want practical ways to move faster without rewriting everything, this event is for you.

On **June 16, 2026 (4:00 PM–8:00 PM UTC)**, Microsoft Reactor is hosting **.NET Day on Agentic Modernization**—a focused livestream for engineering teams looking to reduce legacy maintenance, accelerate cloud migration, and add AI-driven capabilities to existing applications.

[Register Now](https://developer.microsoft.com/reactor/events/27243/)

## Why attend

You’ll learn how to:

- Modernize legacy .NET apps with **GitHub Copilot**-assisted workflows
- Build **agentic experiences** with Microsoft Agent Framework and Microsoft Foundry
- Automate migration and deployment tasks using **GitHub Copilot for Azure (Azure MCP)**

This is a hands-on, real-world agenda built for teams balancing delivery pressure with modernization goals.

## Session highlights

Featured speakers include **Mika Dumont, David Pine, Merrie McGaw, Klaus Loeffelmann, Jerry Nixon, Kaushaya Ganguly, Gaurav Seth, Andrew Westgarth, Jordan Selig, and Bruno Capuano**.

You’ll see sessions on:

- Modernizing .NET apps with GitHub Copilot
- Using Aspire to modernize without a full rewrite
- Migrating real-world WinForms and data-heavy line-of-business apps
- Adding agentic functionality and AI-powered experiences after migration

## Start with this Build 2026 session

Before the event, watch this Microsoft Build 2026 session that introduces the same modernization theme:

**Modernize intelligent apps and agents with .NET that scale as you grow (OD801)**

We’ll see you at **.NET Day on Agentic Modernization**.
